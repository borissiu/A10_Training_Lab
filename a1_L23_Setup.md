## Show Layer 2 & 3 Info.
#### 显示 Layer 1 信息
```
show interface brief
show interface ethernet
show interface ve
show interface trunk
show interface media
show interface statistics
```

#### 显示 Layer 2 信息
```
show mac
show arp
show vlan counters
show run vlan
```

#### 显示 Layer 3 信息。
```
show ip route
show run ip route
```
