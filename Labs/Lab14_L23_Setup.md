## Show commands
#### 显示 Hardware 信息
```
show hardware
```

#### 显示 software version 信息
```
show version
```

#### 显示 CPU 信息
```
show cpu
show cpu history
```

#### 显示 memory 信息
```
show memory
```

#### 显示 Disk 信息
```
show disk
```

#### 显示 Environment 信息
```
show environment
```

#### 显示 Layer 1 信息
```
show interface brief
show interface ethernet
show interface ve
show interface trunk
show interface media
show interface statistics
```

#### 显示 Layer 2 信息
```
show mac
show arp
show vlan counters
show run vlan
```

#### 显示 Layer 3 信息
```
show run ip route
show ip route
```

#### 显示 SSH/GUI/AxAPI 信息
```
show admin session
clear admin session all
```

#### 显示 bootimage & startup-config 信息
```
show bootimage
show startup-config all
show startup-config
show startup-config all-partition
```

#### 显示 syslog 信息

show run logging

clear log
```

#### 显示 audit 信息
```
show audit logging
clear audit log
```

```
show snmp stats
